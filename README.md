# Fazbear Collector - Legal & Support Site

Static legal and support pages for the [Fazbear Collector](https://github.com/drewTuzson/fazbear-collector-legal) iOS app.

Hosted via GitHub Pages at:
- Landing: https://drewtuzson.github.io/fazbear-collector-legal/
- Privacy Policy: https://drewtuzson.github.io/fazbear-collector-legal/privacy.html
- Terms of Use: https://drewtuzson.github.io/fazbear-collector-legal/terms.html
- Support: https://drewtuzson.github.io/fazbear-collector-legal/support.html

These URLs are referenced from the App Store Connect listing (Privacy Policy URL, Support URL, Terms URL) and from inside the iOS app's Settings screen.

## Editing

Each page is a single self-contained HTML file with inline CSS, no build step. Edit, commit, push to `main`, and GitHub Pages publishes within a minute.

Effective dates are encoded in each page's subtitle. Update the date when making a substantive change.
